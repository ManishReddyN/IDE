# Code Runner

Welcome to the **Code Runner** repository! This project allows users to execute code online directly from the browser. It includes additional features for sharing code execution results via generated shortlinks.

## Purpose

The **Online IDE** is a React-based application that provides an intuitive interface for:
- Writing and executing code on the web.
- Sharing code snippets and their respective input/output via shortlinks.
- Supporting a user-friendly development environment, complemented with a Node.js backend.

## Features
- **Online Code Execution**: Compile and run code seamlessly within the web interface.
- **Shortlink Generation**: Easily share code, input, and output with others via uniquely generated shortlinks.
- **Modern Frontend**: Built with React for a dynamic and responsive user experience.
- **Powerful Backend**: Node.js powers the backend for handling requests and code execution.


## Getting Started

### Prerequisites
- **Node.js**: Install the latest version of Node.js.
- **Package Manager**: npm or yarn installed globally.
- **Git**: Ensure Git is installed for version control.

### Instructions
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/ManishReddyN/IDE.git
   cd IDE
   ```

2. **Install Dependencies** for `frontend` and `backend`:
   ```bash
   # Navigate to the respective directories and install dependencies
   cd frontend
   npm install
   cd ../backend
   npm install
   ```

3. **Start the Development Servers**:
   - Backend:
     ```bash
     cd backend
     npm start
     ```
   - Frontend:
     ```bash
     cd frontend
     npm start
     ```

4. **Access the Application**:
   Open your browser and head to `http://localhost:3000`.
